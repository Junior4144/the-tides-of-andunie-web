# The Tides Of Andunië

_A top-down action RPG set in a Tolkien-inspired world. Defeat the pirates and save the Island of Erressëa._

[Play the WebGL build on itch.io](https://nospoonstudio.itch.io/the-tides-of-andunie)  

---

## Tech Stack & Tools

- **Engine**: Unity (version: 6.2)
- **Language**: C#  
- **Target Platforms**: Windows, MacOS, WebGL 
- **Art Style**: 2D pixel art, top-down perspective
